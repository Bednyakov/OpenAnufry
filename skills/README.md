# Навыки агента

Эта директория содержит модульные навыки для расширения возможностей агента.

## Быстрый старт

### Структура навыка

```
skills/
└── my-skill/
    ├── SKILL.md              # Обязательно: описание и инструкции
    ├── scripts/              # Опционально: исполняемые скрипты
    │   └── script.sh
    └── references/           # Опционально: справочные материалы
        └── cheatsheet.md
```

### Минимальный SKILL.md

```markdown
---
name: my-skill
description: Краткое описание навыка
triggers:
  - "ключевое слово"
  - "фраза для активации"
---

# Инструкции для агента

Подробное описание того, как использовать этот навык...
```

## Доступные навыки

### mongo-compass
**Описание:** Работа с MongoDB через mongosh и MongoDB Compass  
**Триггеры:** mongo, mongodb, compass, "запрос в mongo", "найти в mongodb"  
**Требования:** mongosh или mongo CLI

## Документация

Полная документация по архитектуре системы навыков: [SKILLS_ARCHITECTURE.md](../docs/SKILLS_ARCHITECTURE.md)

## Создание нового навыка

1. Создайте директорию:
   ```bash
   mkdir -p skills/my-skill/{scripts,references}
   ```

2. Создайте `SKILL.md` с YAML frontmatter

3. Перезапустите агента

Навык загрузится автоматически!
